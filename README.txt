このプログラムは3DCGをキャッシュするための
キャッシュ機能を試作したものです。

今のところモデルデータを使っても、
同一であるかの判断が難しいため文字列で試している

モデルデータ(InputStream)を表示するプログラムがないので確かめられない(´･ω･`)

文字列をキャッシュに保存しファイル出力。
その後、ファイルから読み込んだものと元の文字列を比較したとこと同一になった。

データベースとの連携テストとキャッシュの挿入削除のテストを行った
